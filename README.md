# Bozhidar-Chervenkov-employees
This program helps identify the pair of employees who have worked
together on common projects for the longest period of time.

The project uses CSV files as input with the following format:
---- EmpID, ProjectID, DateFrom, DateTo ----

How to start:

1. Console application start -> F5 on console startup. When prompted "Enter CSV file path", type "sample.csv" for the default exercise input or the path to your local file.
2. UI startup -> Set Employees.UI as StartUp project and press F5. Use the UI to Load CSV and select file.
